# NowPay Go v0.2.0 — Full API coverage (all-apis)

**Release date:** 2025-03-13

### Install

```bash
go get github.com/muxover/nowpay-go@v0.2.0
```

## Summary

v0.2.0 aligns the SDK with the official NOWPayments API Postman collection: Auth (JWT), Balance, extended Currencies (full-currencies, merchant coins, fixed_rate), CreateFromInvoice, UpdateMerchantEstimate, MinAmountEx with fiat/fixed-rate options, and full Payouts (validate address, list, min amount, fee, cancel, verify). Invoice and payment list params are extended.

## What's included

- **Client**: `Auth(ctx, email, password)` for JWT; `Balance(ctx)`; optional `Config.Token` for endpoints that require JWT (List payments, Create payout)
- **Currencies**: `Supported`, `SupportedWithFixedRate`, `Available`, `FullCurrencies`, `MerchantCoins`
- **Payments**: `CreateFromInvoice`, `UpdateMerchantEstimate`; list payments with `InvoiceID` param
- **Estimates**: `MinAmountEx` with fiat_equivalent, is_fixed_rate, is_fee_paid_by_user
- **Invoices**: Create request extended with PayCurrency, PartiallyPaidURL, IsFixedRate, IsFeePaidByUser
- **Payouts**: `ValidateAddress`, `List`, `MinAmountForWithdrawal`, `Fee`, `Cancel`, `Verify`
- New models for auth, balance, invoice-payment, min-amount params, payout list/validate
- **Retry**: optional `Config.Retry` for 429 and 5xx with exponential backoff (production-ready)
