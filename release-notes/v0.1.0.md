# NowPay Go v0.1.0 — Initial release

**Release date:** 2025-03-13

### Install

```bash
go get github.com/muxover/nowpay-go@v0.1.0
```

## Summary

First release of NowPay Go: Go SDK for NOWPayments (payments, fiat, webhooks). Covers payments, invoices, currencies, estimates, payouts, subscriptions, webhook verification, API status, payment flow, refunds, batch payouts, and subscription cancel/update.

## What's included

- Client with configurable API key and base URL; API status (health) check
- Payments: create, get by ID, list, get flow (details), refund
- Invoices: create, get
- Currencies: supported and available
- Estimates: price conversion (crypto and fiat: USD, EUR, etc.)
- Payouts: create, get status, batch (mass) create
- Subscriptions: create, get, cancel, update
- Webhook: signature verification and typed event parsing
- Structured errors and context support on all API calls
