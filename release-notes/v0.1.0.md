# NowPay Go v0.1.0 — Initial release

**Release date:** 2025-03-13

### Install

```bash
go get github.com/muxover/nowpay-go@v0.1.0
```

## Summary

First release of NowPay Go: Go SDK for NOWPayments (payments, fiat, webhooks, Telegram). Covers payments, invoices, currencies, estimates, payouts, subscriptions, webhook verification, and Telegram helpers.

## What's included

- Client with configurable API key and base URL
- Payments: create, get by ID, list
- Invoices: create, get
- Currencies: supported and available
- Estimates: price conversion (crypto and fiat: USD, EUR, etc.)
- Payouts: create, get status
- Subscriptions: create and manage
- Webhook: signature verification and typed event parsing
- Telegram: payment buttons and invoice message helpers
- Structured errors and context support on all API calls
